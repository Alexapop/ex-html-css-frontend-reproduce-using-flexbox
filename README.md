# Exercise - HTML5 & CSS3 - Frontend - Reproduce using Flexbox

## Objective
### Learn Flexbox to achieve responsive designs. Reproduce the following image using HTML and CSS. Use lorem ipsum as text.

## Requirements
* Use CSS Flexbox for layouts and responsiveness
* Import and use a custom font from Google Fonts
* Maintain a clean, semantic HTML5 structure
* The design adapts correctly to different screen sizes (desktop and mobile)
* Columns, texts, and blocks do not break


## Technical Implementation
* Global `<header>` for the main application title "HTML & CSS"
* The layout is built semantically using independent `<article>` elements for each card, containing nested `<header>` tags for the title, an image and paragraph
* Style reset using @import "reset.css"
* Web typography loaded using @import Google Fonts
* Selected Google Font Family: Opens Sans, Sans-Serif

## Technologies
* HTML5
* CSS3
